This HTML structure sets up a basic webpage with a product listing section, a shopping cart section, and a total price display.

Step 2: Create the JavaScript File

Create a new JavaScript file named script.js in the same directory as your HTML file. This is where we'll add the shopping cart functionality.

Step 3: Define Products

In your script.js file, define some sample products as JavaScript objects. Each product should have properties like name, price, and an ID.

javascript
const products = [
    { id: 1, name: "Product A", price: 10.00 },
    { id: 2, name: "Product B", price: 15.00 },
    { id: 3, name: "Product C", price: 20.00 },
];

Step 4: Display Products

Next, write JavaScript code to display the products on your webpage. Loop through the products array and create HTML elements to represent each product.

javascript
const productsContainer = document.getElementById("products");

products.forEach(product => {
    const productDiv = document.createElement("div");
    productDiv.innerHTML = `
        <p>${product.name} - $${product.price}</p>
        <button onclick="addToCart(${product.id})">Add to Cart</button>
    `;
    productsContainer.appendChild(productDiv);
});

This code generates product listings and includes an "Add to Cart" button for each product.

Step 5: Add Items to the Cart

Create a function called addToCart to add items to the shopping cart when the "Add to Cart" button is clicked.

javascript
const cart = [];
const cartContainer = document.getElementById("cart");
const totalContainer = document.getElementById("total");

function addToCart(productId) {
    const product = products.find(p => p.id === productId);
    if (product) {
        cart.push(product);
        updateCart();
    }
}

Step 6: Update the Cart

Define a function called updateCart to refresh the shopping cart display whenever items are added.

javascript
function updateCart() {
    cartContainer.innerHTML = "";
    let totalPrice = 0;

    cart.forEach(item => {
        const cartItemDiv = document.createElement("div");
        cartItemDiv.textContent = `${item.name} - $${item.price}`;
        cartContainer.appendChild(cartItemDiv);
        totalPrice += item.price;
    });

    totalContainer.textContent = totalPrice.toFixed(2);
}

Step 7: Test Your Shopping Cart

Now, you can test your simple shopping cart by opening your HTML file in a web browser. You should see products listed on the page, and when you click "Add to Cart," items should appear in the cart with a running total.
